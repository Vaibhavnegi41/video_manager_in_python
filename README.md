📇 Contact Information Manager (SQLite + Python)
A simple yet efficient Contact Information Manager built with Python, SQLite3, and Colorama for colored CLI output.
This project allows you to add, view, update, and delete contacts with a clean and interactive terminal interface.

✨ Features
📜 List all contacts with ID, Name, and Phone number.
➕ Add new contacts easily.
✏️ Update existing contacts by ID.
❌ Delete contacts by ID.

🎨 Beautiful CLI interface using colorama for colored text.

💾 Persistent storage with SQLite3 database.

🛠️ Tech Stack
-Python 3
-SQLite3 (built-in Python library)
-Colorama (for colorful CLI output)
